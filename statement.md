# Practice With Loops Quiz #1

This Java template lets you get started quickly with a simple one-page playground.

```java runnable

public class Main {

    public static void main(String[] args) {

        //Practice With Loops Quiz #1

        System.out.println("Infinite Loop");

        int x = 0;
        while (x < 35) {   
            System.out.println(x);   
            x--;
        }
    }
}
```
