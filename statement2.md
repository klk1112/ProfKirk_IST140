# Practice With Loops Quiz

This Java template lets you get started quickly with a simple one-page playground.

```java runnable

public class Main {

    public static void main(String[] args) {

        //Practice With Loops Quiz #2

        int count = 0;
        while (count != 500) {   
            System.out.print(count);
            System.out.print(" ");   
            count = count + 50;
        }
        
        System.out.print("...");
    }
}
```
